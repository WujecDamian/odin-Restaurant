## Overview

This is a **single-page restaurant website** created as a practice project for **The Odin Project** curriculum (JavaScript path → Restaurant Page lesson).

The goal of the assignment is to build a modern-feeling restaurant homepage **using only vanilla JavaScript** (no frameworks), while practicing:

- Webpack module bundling
- ES6+ JavaScript modules (`import`/`export`)
- Dynamic DOM manipulation
- Tabbed navigation without page reloads
- Component-like architecture in plain JS
- Clean project structure & asset management

The page represents a fictional kebab/fast-food restaurant called **Friends Kebab**.

## Features

- **Tabbed navigation** (Home / Menu / Contact) built entirely with JavaScript
- Content is **dynamically generated** when tabs are clicked (no multi-HTML-file approach)
- Responsive layout (mobile-friendly to some extent)
- Attractive hero section with background image/gradient
- Menu section displaying food items (kebabs, sides, drinks) with prices
- Simple contact section (possibly with fake address, hours, map placeholder or form)
- Clean, colorful design with food-related imagery
